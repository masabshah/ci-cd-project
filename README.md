# Experiment-20: CI/CD Pipeline for Application Deployment

## Objective
To implement CI/CD pipeline using GitHub Actions and Docker.

## Tools Used
- Node.js
- Docker
- GitHub Actions

## Commands
docker build -t ci-cd-backend .
docker run -p 5000:5000 ci-cd-backend

## Output
Application runs on http://localhost:5000

## Conclusion
Successfully implemented CI/CD pipeline.
